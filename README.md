# Michael Jordan Biography Website

A static HTML and CSS website about Michael Jordan's career, achievements, statistics, awards, and legacy.

## Features

- Semantic HTML5 structure
- Responsive-friendly layout
- Dark theme
- Table of contents with navigation
- Career statistics tables
- Awards and achievements
- Accessible navigation and images

## Technologies Used

- HTML5
- CSS3
